 ### Passo a Passo
 
 - Instalar o nodejs
 - Instalar o npm
 - Criar diretorio do projeto e executar o comando:
 ```
 npm init -y
 ```
 - Instalar as dependencias
 ```
 npm install http express debug --save
 ```

 - Instalar o mongoose
 ```
 npm install mongoose
 ```

 - Criar conta no MATLAB (mongoDB)
